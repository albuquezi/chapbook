Chapbook, um formato de história para o Twine 2

![][logo.svg]

de Chris Klimas

Chapbook é um formato de história para o Twine 2, o que quer dizer que permite que se joguem histórias criadas no Twine, num navegador. Para o usares, adiciona este endereço como um formato de história:

https://klembot.github.io/chapbook/use/1.0.0-beta1/format.js

O Chapbook está ainda em estado beta. Produz histórias funcionais, mas ainda está pouco trabalhado, e é provavel que tenha "bugs".

O Chapbook foi lançado sob a licença MIT. Pode ser usado para criar trabalhos comerciais sem necessidade de pagamento de direitos de autor ou de qualquer outro tipo de acordo.

  • Aprender mais sobre o Chapbook

    O guia é uma suave introdução à escrita com o Chapbook. Começa por aqui se quiseres criar histórias com o Chapbook.

  • O Manto da Escuridão, uma história de demonstração

    Queres ver rapidamente como são as histórias feitas com o Chapbook ou como é que se programam? Dá uma olhada a esta história e ao seu código-fonte em formato Twee. Se não sabes o que é o Twee, é um formato de texto compacto para as histórias do Twine. A coisa principal a reter é que cada linha que comece com :: descreve uma nova passagem.

  • Comunica um "bug" ou sugere uma funcionalidade

    Neste momento, a comunicação de "bugs" e sugestão de novas funcionalidades está limitada às pessoas que apoiam o projeto no Patreon. O sistema habitual de comunicar problemas será aberto quando o Chapbook 1.0 for lançado.

  • Apoia o desenvolvimento do Chapbook

    Apoia o projeto no Patreon para teres acesso aos diários de desenvolvimento e a outros benefícios.
